JS Intermediate

  • Booleanos y condicionales. Cohersión.
  • Loops: for, while, forEach, Array.map, Array.filter
  • Arrow functions. Hosting en JS.
  • El scope de las Arrow Functions siempre es windows
  • Destructuración en objetos y arrays
  • Spread operator para separar en la destructuración ...
  • Spread operator no deja referencias, son copias inmutables, pero superficiales, no interiores
  • Librería lodash para hacer copias profundas de objetos y arrays: cloneDeep()
  • Lodash se puede reemplazar con JSON.parse(JSON.stringify(value))
  • Prototipos en JS. Es lo más parecido a las clases, de otros programas.
  • Prototipo utiliza una FUNCTION con THIS para crear propiedades y métodos y NEW para crear instancias.
  • Coerción de tipos, utilidad de JS por ser de tipado blando, que funciona en tiempo de ejecución.
